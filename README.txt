To run this program:

1.	Unzip the File

2.	Go to Environment Variables (in Control Panel) and make sure Java JDK is added to the Path variable
	*if not, click on Path, then Add, then enter the path for java. it should be simular to this:	
	
					C:\Program Files\Java\jdk-13.0.1\bin

3.	Double-click the Comand Script called "Run Me" in the "target" folder
